# PitchPal AI

Interactive prototype for an AI-powered entrepreneurship coaching platform for schools.

## Run locally

```bash
npm install
npm run dev
```

The prototype simulates investor questions, live scoring, feedback, teacher analytics, and classroom competition voting without requiring an AI API key.

## AI-powered modules

- AI Pitch Coach for student practice feedback.
- AI Investor Panel for finance, sustainability, marketing, and innovation questions.
- AI Rubric Scoring for clarity, confidence, innovation, and financial understanding.
- AI Teacher Insights for class summaries and next-lesson recommendations.
- AI Competition Judge for pitch-day scoring support.
- AI SDG Mapper for connecting student ideas to impact goals.
